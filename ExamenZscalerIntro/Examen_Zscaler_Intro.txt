1. Zscaler EDU-200 - Introducción a Zero Trust
1.1. What is a critical flaw in the Traditional Perimeter Network Security Architecture as highlighted in the content?
1.1.a. Limited access for remote workers through VPN
1.1.b.*. Implicit trust of users once they cross the perimeter
1.1.c. Over-reliance on internal DC/Colo for resource management
1.1.d. Lack of centralized Firewall/VPN for defense

1.2. What is a core principle of the Zero Trust security model implemented by Zscaler, and why is it necessary for modern cloud-driven environments?
1.2.a. Extend implicit trust to all devices connected through VPNs and firewalls
1.2.b. Grant broad network access based on user location and device type
1.2.c.*. Never trust, always verify—access is granted based on identity and context, ensuring every connection is dynamically validated regardless of network or device
1.2.d. Allow full lateral movement within the network for ease of collaboration

1.3. Why is traditional perimeter security considered obsolete in a cloud-driven remote work environment?
1.3.a. Because users no longer need access to applications
1.3.b.*. Because users, applications, and data are no longer limited to a single corporate network perimeter
1.3.c. Because firewalls are no longer able to block any traffic
1.3.d. Because all modern applications must be public
